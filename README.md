🌍 Почему MiniArch?
Потому что твой компьютер — твои правила.

Никакой телеметрии. Никакого "мы обнаружили подозрительную активность". Никаких "системных требований". Только ты и код, который ты можешь прочитать, изменить, улучшить.

MiniArch — это не просто "ещё одна поделка на Python". Это манифест. Это способ показать, что операционная система может быть:
- Прозрачной — весь код перед твоими глазами
- Лёгкой — работает на старом железе
- Твоей — ты решаешь, что в ней будет

🎯 Для кого это

- Для тех, кого тошнит от Windows с её рекламой и слежкой
- Для тех, кто хочет понять, а не просто нажимать кнопки
- Для тех, кто ценит природу — минимум ресурсов, максимум эффективности
- Для тех, кто готовится к настоящему Linux (Arch, Gentoo, LFS)
- Для тех, кто верит, что знания должны быть свободными

🚀 Быстрый старт

1. Скачай
git clone https://github.com/matvpro2010/miniarch.git
cd miniarch


2. Установи зависимости
pip install psutil  


### 3. Запусти
python miniarch.py

4. Введи `help`
[user@miniarch ~]$ help

🌱 Экологичность

MiniArch создана с уважением к планете:
- 0 лишнего кода — не использует CO2 на серверах потому что их у нас нет
- Работает на старом железе — меньше электроники на свалке
- Нет телеметрии — не жрёт трафик
- Open Source — не плодит проприетарный мусор
